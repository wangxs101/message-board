Setup Required: Create a free JSONBin to store messages.
1. Go to jsonbin.io and sign up (free)
2. Create a new Bin with initial content: {"messages":[]}
3. Copy your Bin ID and Master Key
4. Set them below:
const BIN_ID = 'YOUR_BIN_ID';
const API_KEY = 'YOUR_API_KEY';
